# Sistema de Reservas – Cafetería Universitaria

Proyecto sencillo desarrollado en C# como parte de la Evaluación Final.
Usa matrices, funciones y menú interactivo para gestionar reservas.

## Funcionalidades
- Mostrar menú de combos  
- Registrar reservas  
- Cancelar reservas  
- Listar reservas por turno  
- Buscar por nombre  
- Reporte de ingresos  

## Estructura del Proyecto
- `Program.cs` — contiene toda la lógica del sistema.

## Ejecución
1. Abrir el proyecto en Visual Studio o VS Code.
2. Ejecutar `Program.cs`.
3. Navegar con el menú del sistema.

## Ejemplo de uso
  Ver menú de combos
  Registrar reserva
  Ingrese nombre: Ana
  Seleccione combo: 1
  Reserva registrada.
